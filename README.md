# Movie app in JS

## Configure settings

```
Create an .env file in the root and add the following:
VUE_APP_TITLE={name of my application}
VUE_APP_API_KEY={key}
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
